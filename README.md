# Vehicle Parking Management System

A full-stack web application built using Flask, SQLite, HTML, CSS, Bootstrap, and SQLAlchemy to streamline vehicle parking operations through efficient parking slot allocation, booking, and management.

## User Roles

### Administrator

* Manage parking lots and parking spots
* Monitor parking occupancy and availability
* View registered users and parking records
* Track parking activity through the admin dashboard
* Manage overall parking operations

### User

* Register and log in securely
* View available parking spaces
* Book and reserve parking spots
* Park and release vehicles
* View booking history and parking status

## Features

* User Registration and Authentication
* Parking Slot Allocation and Management
* Vehicle Parking Reservation System
* Real-Time Parking Availability Tracking
* Admin Dashboard for Monitoring and Control
* Booking History and Status Tracking
* Secure Database Management

## Technologies Used

* Python
* Flask
* SQLite
* SQLAlchemy
* HTML
* CSS
* Bootstrap

## Project Objective

The Vehicle Parking Management System is designed to simplify and automate parking operations by providing a user-friendly platform for vehicle owners to reserve parking spaces and for administrators to efficiently manage parking facilities. The application improves parking space utilization, reduces manual effort, and enhances the overall parking experience.
